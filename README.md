# anypool

General-purpose connection pooling library.

## WARNING

This software is still ALPHA quality. The APIs will be likely to change.

## API

- make-pool
- pool-max-open-count
- pool-max-idle-count
- pool-open-count
- pool-active-count
- pool-idle-count
- fetch
- putback
